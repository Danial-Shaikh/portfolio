006 / MOBILE · IN PROGRESS
iGotchu in progress

iGotchu — A real-time family location-sharing app inspired by Life360. Built as a fully self-contained web app, iGotchu puts every member of your circle on a live map, lets you save trusted places like home and school, and fires instant arrive/leave alerts the moment someone enters or exits a geofenced zone. Features live GPS tracking via the Geolocation API, an interactive map, battery and last-seen status, and a one-tap SOS. Built with vanilla JavaScript, Leaflet, and the browser Geolocation API — zero backend, fully client-side.
